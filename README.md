## 2.项目介绍

### package

- aspect:日志切面类
- bo:查询条件对象
- config:配置类
- constants:常量
- controller:MVC控制器
- dao:Mapper接口
- exception:异常相关
- po:实体类
- security:权限控制相关
- service:业务
  - base:对数据库的crud接口
  - impl:接口实现类
  - ext:扩展接口
- utils:工具类
- vo:接口传输对象`注意:接口请求和接口响应包括但不限于在vo之内`
  - request:接口请求
  - response:接口响应
